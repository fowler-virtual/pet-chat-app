# UX / IA Strategy

## Target Hypothesis

### Core target

- 28歳から45歳の犬猫オーナー
- ペットを家族として扱う
- 写真や動画は撮るが、記録アプリは続きにくい
- AI を使いたいのではなく、うちの子とつながりたい
- 夜、仕事の合間、外出中に短くアプリを開く

### Motivation

- 今どうしているかを感じたい
- 留守番中のさみしさを埋めたい
- 多頭飼いでもそれぞれの子と向き合いたい
- 写真だけでなく、関係性が積み上がる体験がほしい

## Product Principle

- 管理より感情接続を優先する
- AI を見せず、ペット本人感を見せる
- 最初の1分で会話を始められるようにする
- 続けるほど愛着が増す構造にする

## New IA

### Today

- 今日のその子
- ひとこと
- 気分
- 直近の会話
- すぐ話す導線

### Talk

- 1対1チャット
- 話しかけチップ
- 入力欄
- 送受信ログ

### Pets

- 多頭飼い管理
- ペット切替
- ペット追加
- ペット詳細編集

### Memory

- 会話から生まれた思い出
- 直近ログの要約
- 関係の蓄積

### Settings

- 認証
- プラン
- 通知
- アプリ設定

## First-Time Experience

初回登録は 3 ステップを基本とする。

1. 写真を選ぶ
2. 名前と種類を入れる
3. 性格を選ぶ

性別や詳細設定は後から編集させる。

## Retention Hooks

- Today で毎日違うひとことを見せる
- Talk に話しかけチップを置く
- Memory で蓄積を見せる
- 2匹目追加や履歴拡張のタイミングで課金を見せる
