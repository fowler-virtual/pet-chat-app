# プライバシーポリシー

最終更新日: 2026年3月9日

「かいぬしとおはなし」（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリが収集・利用・保管する情報について説明します。

## 1. 収集する情報

### 1.1 ユーザーが提供する情報
- **メールアドレス**: データの引き継ぎ（ログイン）機能を利用する場合に取得します。ログインは任意であり、本アプリの基本機能はログインなしでご利用いただけます。
- **ペットのプロフィール情報**: 名前、種類、性別、性格、口調などの設定情報。
- **会話データ**: ペットとの会話内容。
- **画像データ**: ペットのアイコンとしてアップロードされた画像。

### 1.2 自動的に収集する情報
- **利用状況データ**: ログイン日時、会話回数、アプリの起動状況。
- **デバイス情報**: OS種別、アプリバージョン（アプリの動作改善のために使用）。

### 1.3 広告に関する情報
- 本アプリでは広告を表示する場合があります。広告パートナー（Google AdMob等）が独自にデバイス識別子等を収集する場合があります。詳細は各広告パートナーのプライバシーポリシーをご確認ください。

## 2. 情報の利用目的

収集した情報は、以下の目的で利用します。

- 本アプリの機能提供（ペットとの会話、プロフィール管理）
- データのバックアップおよび機種変更時の引き継ぎ
- AI による会話応答の生成
- サービスの改善および不具合の修正
- サブスクリプション管理
- 広告の表示

## 3. 情報の第三者提供

ユーザーの個人情報を、以下の場合を除き第三者に提供することはありません。

- **AI サービスプロバイダー**: 会話応答の生成のため、会話内容を OpenAI の API に送信します。送信されるデータにメールアドレス等の個人を特定する情報は含まれません。
- **広告パートナー**: 広告表示のため、匿名化されたデバイス情報を共有する場合があります。
- **法令に基づく場合**: 法令の要請により開示が必要な場合。

## 4. データの保管

- **ローカルデータ**: ログインしない場合、すべてのデータはお使いの端末内にのみ保存されます。
- **サーバーデータ**: ログインした場合、ペットのプロフィールと会話履歴がサーバーに保存されます。
- **保管期間**: アカウント削除の申請があった場合、30日以内にサーバー上のデータを削除します。

## 5. データの削除

- ペットのデータは、アプリ内の「お別れする」機能でいつでも削除できます。
- ローカルデータは、端末のアプリデータ削除またはアプリのアンインストールにより削除されます。
- サーバー上の全データの削除を希望される場合は、お問い合わせ先までご連絡ください。

## 6. セキュリティ

ユーザーの情報を保護するため、通信の暗号化（HTTPS）およびアクセス制御を実施しています。ただし、インターネットを介した完全なセキュリティを保証するものではありません。

## 7. 未成年者の利用

本アプリは、13歳未満のお子様を対象としたサービスではありません。13歳未満の方が個人情報を提供されたことが判明した場合、速やかに削除いたします。

## 8. ポリシーの変更

本プライバシーポリシーは、サービスの変更に伴い更新される場合があります。重要な変更がある場合は、アプリ内でお知らせします。
